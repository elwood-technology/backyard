Quick Start

1. Create a workspace#

  • TypeScript
  • JavaScript
yarn create backyard ./workspace core-aws-typescript
yarn create backyard ./workspace core-aws

2. Start the workspace#

cd workspace;
yarn backyard local start

4. Make a request#

Make a request to the gateway to make sure it's running.

curl http://localhost:8000/health

Response

{"message":"awesome :)"}

5. Deploy it to AWS#

yarn backyard remote deploy --build
